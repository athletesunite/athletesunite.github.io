Move Up

View Source Code

Move Up was my team's submission at HackDuke. The app uses an interactive county map of the United States to help search for areas with the highest social mobility. The app also displays information on affordable housing options in the area. It was built using Esri API, React, and Leaflet.JS.

We won best use of the Esri Platform.

Clean Water Crowdsourcing

View Source Code

This application helps users allocate safe and clean water resources within their local communities through crowdsourcing. This app was written in Android using the Google Maps and GraphView API, complete with login system and manager/worker support tools.
I loved this app so much it was also made into a desktop application. Source code can be found here!
